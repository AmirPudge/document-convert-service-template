# Document Convert Service Template

## О проекте

Document Convert Service — это микросервис для автоматической конвертации документов в PDF формат.

## Назначение

Сервис предназначен для:

- **Обработки запросов на генерацию PDF**: Принимает поступающие запросы на создание PDF-документов из различных источников
- **Конвертации документов**: Преобразует исходные документы в PDF-формат
- **Сохранения результатов**: Сохраняет сгенерированные PDF-файлы в облачное хранилище для последующего доступа
- **Отслеживания статуса**: Ведет историю обработки запросов и сохраняет информацию о каждом преобразовании
- **Гарантии надежности**: Обеспечивает идемпотентную обработку, то есть каждый запрос обрабатывается ровно один раз, даже при повторных попытках
- **Интеграции с бизнес-процессами**: Уведомляет внешние системы об успешном завершении конвертации

## workflow

1. Сервис получает запрос на конвертацию документа
2. Проверяет, не был ли этот запрос уже обработан ранее
3. Конвертирует документ в PDF
4. Загружает результат в облачное хранилище
5. Сохраняет информацию об обработке в базе данных
6. Уведомляет систему об успешном завершении
